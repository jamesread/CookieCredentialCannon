# CookieCredentialCannon

Assign credentials to people (optionally by firing them via a canon), everyone gets a UUID assigned via a cookie.

Useful for workshops.

## config.yaml

```
server: "http://example.com"
cookieName: "ccc"
credentials:
  - username: user1
    password: user1

  - username: user2
    password: user2

  - username: user3
    password: user3
```
